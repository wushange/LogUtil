# LogUtil
提取xutil的logutil

使用方法  ， 在application中

       LogUtil.setDebug(true);
在代码中使用

	    LogUtil.d("提取xutil的logutil");
        LogUtil.e("提取xutil的logutil");
        LogUtil.i("提取xutil的logutil");
        LogUtil.v("提取xutil的logutil");
        LogUtil.w("提取xutil的logutil");
        LogUtil.wtf("提取xutil的logutil");
